# Information
All source files should be compiled with gcc with the following flags, and should work with either standards of c89 or c99:
```sh
$ gcc -O -Wall -W -pedantic -ansi -std=c89 filename.c -o filename
```

```sh
$ gcc -O -Wall -W -pedantic -ansi -std=c89 filename.c -o filename
```
