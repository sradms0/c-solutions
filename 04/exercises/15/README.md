### Exercise 4.15
Give the values of `i` and `j` after each of the following expression statements has been executed. (Assume that `i` has the value 1 intially and `j` has the value 2.)  
(a) `i += j;`  
(b) `i--;`  
(c) `i * j / 1;`  
(d) `i % ++j`;  

### Solution
(a) 3 2  
(b) 2 2  
(c) 2 2  
(d) 2 3
